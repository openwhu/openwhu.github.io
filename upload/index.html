    上传须知|Rules¶

    1.所有资料必须为任课教师向学生公开内容¶

    2.建议上传的类别¶

    笔记、历年试卷、攻略与评价、公开课件、作业或实验范例、复习资料、学习经验

    3.建议文件名命名格式¶

    以课程名命名即可

    4.建议文件格式及组织¶

    课程评价：可写在对应课程文件夹的 README.md中

    上课时间及教师：可写在README.md中

    其他文件（如课件资料）：PDF格式

    6.其他注意事项:¶

    单次pull request避免上传多门课资料，否则因Github只能review前3000文件，不便检查

    请尽量不上传二进制文件（如果有必要请以压缩包形式上传）

    上传方式|Method¶

    1.通过Pull Requests¶

    由于本项目体积很大，故推荐采用在GitHub Web端直接上传的方式

    如有疑问可参考：知乎：如何使用github

    具体操作如下：

    查看详情|View Details 1. Fork 本项目

    2. 上传文件到已有文件夹/新建文件夹后上传文件

    3. 提交 Pull Request

    2.通过Issue¶

    或者也可以直接附加在Issue中，由维护者进行添加

    3.通过邮件¶

    或者也可以发送邮件至 OpenWHU@outlook.com ，由维护者进行添加
